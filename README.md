# 🔐 MERN Authentication System with Email OTP

A full-stack authentication system built using the MERN stack (MongoDB, Express.js, React.js, Node.js) with secure email OTP verification.

---

## 🚀 Features

- User Registration with Email OTP Verification  
- Secure Login using JWT  
- Protected Routes  
- User Dashboard  
- Forgot Password with OTP  
- Password Reset Functionality  

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Axios
- CSS / Tailwind / Bootstrap

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (MongoDB Atlas)

**Other Tools:**
- NodeMailer (Email OTP)
- JWT (Authentication)
